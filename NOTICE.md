# Third-party software

ProtoForge is a workflow/skill that **drives** the following independent tools. It does not
bundle or redistribute them — your friends install them separately under their own licenses:

- **KiCad** — GPLv3 (https://www.kicad.org)
- **FreeCAD** — LGPLv2+ (https://www.freecad.org)
- **Blender** — GPLv3 (https://www.blender.org)
- **Freerouting** — GPLv3 (https://github.com/freerouting/freerouting)
- **KiPilot MCP**, **freecad-mcp** (neka-nat), **blender-mcp** (ahujasid) — see each project's license.
- **Claude Code** / Anthropic API — subject to Anthropic's terms.

If you ever package, host, or redistribute any of these (rather than just driving a local copy),
review each project's license obligations first.
